Place your background image here named 'app-bg.png' (or set REACT_APP_DEFAULT_BG_URL).

By default the app looks for:
  /backgrounds/app-bg.png

Optional: set a custom URL in a .env file at project root:
  REACT_APP_DEFAULT_BG_URL=https://your.cdn/image.jpg

Tip: Use a wide 16:9 image (1920x1080 or larger) for best results.
